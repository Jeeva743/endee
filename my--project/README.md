# 🤖 AI FAQ Chatbot using Vector Search

## 📌 Overview
This project is an AI-based FAQ chatbot that answers user questions using similarity search.

## 🚀 Features
- Converts text into vectors using TF-IDF
- Uses cosine similarity to find best match
- Simple and fast AI chatbot

## 🧠 How it works (Vector Concept)
- Questions are converted into vectors
- User input is also converted into a vector
- Cosine similarity is used to compare vectors
- The most similar question is selected and its answer is returned
